### 9. **Debouncing & Throttling**
- **src/ListPanel.tsx**:  
  - **Severity**: LOW  
  - **Dimension**: Debouncing & Throttling  
  - **Issue**: Keyboard navigation handler in `useInput` fires on every keypress; rapid key repeats could cause excessive state updates and `onSelectItem` calls.  
  - **Fix**:  
    **Before:**  
    ```ts
    useInput(
      (input: string, key: Key) => {
        // ... sets selection on every keypress
      },
      { isActive: isFocused }
    );
    ```
    **After:**  
    ```ts
    // Add a debounce to navigation handler
    import debounce from 'lodash.debounce';
    const handleInput = debounce((input: string, key: Key) => {
      // ... same logic
    }, 50, { leading: true, trailing: false });

    useInput(handleInput, { isActive: isFocused });
    ```
  - **Impact**: Prevents UI lag and excessive re-renders on rapid key repeats, improving UX responsiveness.
