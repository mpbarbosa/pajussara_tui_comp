### 4. Memory Leaks

**File:** `demos/listpanel-cities.tsx` (and similar: `listpanel-cities2.tsx`, `status-chronometer-cities*.tsx`)  
**Severity:** MEDIUM  
**Dimension:** Memory Leaks  
**Issue:** Timers (`setTimeout`) in `useEffect` are not always cleared if the component unmounts during a running tour.  
**Fix:**  
_Before:_  
```ts
useEffect(() => {
  if (tourDone) return;
  const timer = setTimeout(() => { ... }, VISIT_MS);
  return () => clearTimeout(timer);
}, [tourIndex, tourDone]);
```
_After:_  
```ts
useEffect(() => {
  if (tourDone) return;
  let cancelled = false;
  const timer = setTimeout(() => {
    if (!cancelled) { ... }
  }, VISIT_MS);
  return () => {
    cancelled = true;
    clearTimeout(timer);
  };
}, [tourIndex, tourDone]);
```
**Impact:**  
If the component unmounts while a timer is pending, state updates may be attempted on an unmounted component, risking memory leaks or React warnings.
