### 4. Memory Leaks

**File:** demo/listpanel-cities.tsx  
**Severity:** MEDIUM  
**Dimension:** Memory Leaks  
**Issue:** `setTimeout` in `useEffect` (tour timer) is correctly cleared, but if the component is unmounted while a timer is pending, the timer is cleared but any in-flight state updates (setDurations, setTourIndex) may still run, potentially causing React "setState on unmounted component" warnings.  
**Fix:**  
_Before:_  
```js
return () => clearTimeout(timer);
```
_After:_  
```js
let isMounted = true;
const timer = setTimeout(() => {
  if (!isMounted) return;
  // ...existing code...
}, VISIT_MS);
return () => {
  isMounted = false;
  clearTimeout(timer);
};
```
**Impact:**  
Prevents state updates on unmounted components, avoiding memory leaks and React warnings.
