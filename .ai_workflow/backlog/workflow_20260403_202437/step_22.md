## Operable

### src/ListPanel.tsx
- **Issue:** No visible focus indicator for keyboard navigation (tab/arrow keys).
  - **WCAG 2.4.7 (Focus Visible)** — **Severity:** Major
  - **Remediation:** Ensure the currently selected/focused item is visually distinct (e.g., via color, underline, or symbol).  
    _Example:_  
    ```tsx
    <Text inverse={isFocused && selIdx === i}>...</Text>
    ```
    Or, ensure a visible marker (e.g., `>`) is always present for the focused item.

- **Issue:** No explicit ARIA roles or properties for list structure.
  - **WCAG 4.1.2 (Name, Role, Value)** — **Severity:** Minor
  - **Remediation:** Add `role="list"` to the container and `role="listitem"` to each item for screen reader clarity.
    _Example:_
    ```tsx
    <Box role="list">...</Box>
    <Text role="listitem">...</Text>
    ```

### src/StreamViewer.tsx
- **Issue:** No explicit ARIA live region for dynamic AI output.
  - **WCAG 4.1.3 (Status Messages)** — **Severity:** Major
  - **Remediation:** Add `aria-live="polite"` to the main output area so screen readers announce updates.
    _Example:_
    ```tsx
    <Box aria-live="polite">...</Box>
    ```
